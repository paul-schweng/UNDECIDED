# 1 Use-Case Name
Edit Profile

## 1.1 Brief Description
A user is able to make changes to his profile attributes, like his name or email etc.
# 2 Flow of Events
## 2.1 Basic Flow

- User clicks on "edit profile" button
- User changes either 
    - his email
    - his description
    - his name
    - his new password
    - his profile picture
    - nothing
- User clicks on "save changes" button
